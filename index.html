NHS-ui
A Simple, consistant front-end framework
About

NHS styles is built to be an alternative to front end css frameworks.

The main advantages of this framework are;

  • Easy to use, minimal code, learning curve
  • Consistant
  • Lean, reducing complexity in UI and acheiving faster page load
  • Conforms with NHS Identity Guidelines

Bolier Plate

Start with this. The Boiler plate is the layout for your page.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible"content="IE=edge">
<meta name="viewport"content="width=device-width, initial-scale=1.0">
<link rel="stylesheet"href="https://nhs-ui.pages.dev/nhs.css">
<title>Home - SystemName</title>
</head>
<body>
 
<header class="bg-blue-4 text-white">
<div class="container flex">
<div class="block logo">
 
<img src="nhswales-gold.png"alt="SystemName Logo">
<a href="/">SystemName</a>
</div>
 
<div class="block search">
<input type="search">
<a class="button"type="submit"value="Search"><i class="fas fa-search"></i></a>
</div>

<nav class="block text-right text-middle">
<a href="/"><i class="fas fa-home"></i></a>
<a href="/settings"><i class="fas fa-cog"></i></a>
<a href="/logout"><i class="fas fa-sign-out-alt"></i></a>
</nav>
</div>
</header>

<main>
<h1 class="text-blue">Title</h1>
<p>Main content area</p>
</main>

<footer class="bg-grey-1 text-center">
<a href="/">SystemName</a> - Footer text
</footer>

</body>
</html>

Layout

Container

Max width, centered on page

A conatiner has a max-width of x and is centered on the page. You only need to use this to stop content filling the screen width.

If you are using the boiler plate you shouldn't need to add any additional containers.

<div class="container">
...
</div>

Block 2
Block 3

A block sits inside container to create spaces for content.
